1. What is jQuery?
jQuery is a fast, small, and feature-rich JavaScript library. It simplifies tasks like HTML DOM traversal, manipulation, event handling, and animation.
2. How do I include jQuery in my project?
You can include jQuery using a CDN like Google or jQuery's official CDN. Example: <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
3. What does the jQuery toggle() function do?
The toggle() method shows or hides an element with a sliding or fading effect, depending on the method used (e.g., slideToggle, fadeToggle).
4. What is the difference between .hide() and .toggle() in jQuery?
.hide() hides the element, whereas .toggle() toggles between showing and hiding the element.
5. Can jQuery effects be chained?
Yes, jQuery allows method chaining, so you can chain multiple effects or functions on a single element.
